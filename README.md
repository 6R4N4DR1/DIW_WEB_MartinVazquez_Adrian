# 6R4N4DR1 Web: PHANTOM Central Hub

This is the web project "PHANTOM Central Hub" for the Web Interface Design (DIW) course of the Web Application Development (DAW) program. This project has been developed by me, Adrian Martin Vazquez.

## Description

The "PHANTOM Central Hub" project is a centralized platform that serves as a hub for various activities and events of the PHANTOM community. The main objective is to provide an intuitive and attractive user interface for end users.

## Installation

To install and run this project locally, follow these steps:

1. Clone the repository:

```sh
   git clone https://github.com/6R4N4DR1/DIW_WEB_MartinVazquez_Adrian.git
```

2. Navigate to the project directory:

```sh
   cd DIW_WEB_MartinVazquez_Adrian
```

3. Install the dependencies:

   ```sh
   npm install
   ```
4. Compile the SCSS files to CSS:

   ```sh
   npm run sass
   ```
